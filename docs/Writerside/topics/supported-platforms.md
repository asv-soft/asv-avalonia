# Supported Platforms

| Platform    | Status                                                                                      | Notes                                                          |
|:------------|:--------------------------------------------------------------------------------------------|:---------------------------------------------------------------|
| **Windows** | ![Windows](https://img.shields.io/badge/Windows-Fully%20Supported-brightgreen?logo=windows) | Primary platform; supports full functionality.                 |
| **Linux**   | ![Linux](https://img.shields.io/badge/Linux-Supported-yellow?logo=linux)                    | **Known Issue:** Custom window control buttons may not appear. |
| **macOS**   | ![macOS](https://img.shields.io/badge/macOS-Supported-yellow?logo=apple)                    | Generally supported with minor known issues.                   |
| **Mobile**  | ![Mobile](https://img.shields.io/badge/iOS%20%2F%20Android-Not%20Tested-lightgrey)          | Currently untested.                                            |

Encounter any platform-specific bugs? Please [open an issue](https://github.com/asv-soft/asv-avalonia/issues).
