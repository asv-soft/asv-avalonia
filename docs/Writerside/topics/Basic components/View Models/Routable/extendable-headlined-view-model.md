# Extendable Headlined View Model

[`ExtendableHeadlinedViewModel`](https://github.com/asv-soft/asv-avalonia/blob/main/src/Asv.Avalonia/ViewModel/Extendable/ExtendableHeadlinedViewModel.cs) is a [`HeadlinedViewModel`](headlined-view-model.md) that supports extensibility via [`ExtendableViewModel`](extendable-view-model.md).
It derives from `ExtendableViewModel` and implements `IHeadlinedViewModel`.
