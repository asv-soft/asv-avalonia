# Extendable Tree Subpage

[`ExtendableTreeSubpage`](https://github.com/asv-soft/asv-avalonia/blob/main/src/Asv.Avalonia/Controls/TreePage/TreeSubpage/ExtendableTreeSubpage.cs) is a [`TreeSubpage`](headlined-view-model.md) that supports extensibility via [`ExtendableViewModel`](extendable-view-model.md).
It derives from `ExtendableViewModel` and implements `ITreeSubpage`.
