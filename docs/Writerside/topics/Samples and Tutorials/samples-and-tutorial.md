### How to create a module

The framework can be extended with custom [modules](what-is-a-module.md).
This guide explains how to create a simple module.
It covers the following topics:
- Creating a builder;
- Adding a module information file;
- Creating basic components;
- Packaging the module into a NuGet package;
- Using the module in an app alongside other modules.

This guide is the best way for developers to learn the basics of creating a module for Asv.Avalonia.
Follow the guide [here](how-to-create-module.md)
